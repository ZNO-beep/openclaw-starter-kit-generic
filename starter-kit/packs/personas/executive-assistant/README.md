# Persona Pack: Executive Assistant

A generic executive-assistant persona overlay.

## Goals
- concise, action-oriented communication
- proactive but not spammy
- structured decision-making
- strong privacy boundaries

## Install
```bash
bash starter-kit/packs/personas/executive-assistant/scripts/install.sh
```

## Validate

```bash
bash starter-kit/packs/personas/executive-assistant/scripts/validate.sh
```
